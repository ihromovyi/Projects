# The Formula of a Spotify Hit

Analysis of **8,577 tracks** (2021–2025 releases) from the Spotify Global Music Dataset — testing which factors actually drive a track's popularity.

**Verdict (correlations, not guarantees):** a "hit-shaped" track is released recently, sits on an **album** (not a single), runs **3–4 minutes**, is **explicit**, leans **pop / reggaeton / K-pop / alt-pop / R&B**, and comes from an already-popular artist.

**Honest catch:** even popular artists have flops — the data describes the *shape of the streaming market* more than a guaranteed recipe.
